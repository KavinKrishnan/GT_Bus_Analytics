DESCRIPTION

This readme describes essentially how to run the backend Flask server for the Georgia Tech Bus route website created to view waiting times and planning routes.

INSTALLATION

Require at least Python 3.6:
-	Need following libraries installed as well:  Panda,  Flask, Flask_Cors, BS4, numpy

EXECUTION 

To run server:

cd server
python3 ./server.py

----
open index.html in the root folder using FireFox 59

To plan a trip:
1.press start
2. click your starting point in the map
3.press Destination
4. click your Destination point in the map
5. press “Calculate route”

To see the Statistics

1.	Press Statistics
2.	Select a route
3.	For expected waiting time: click a stop in the map.
